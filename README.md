## 🎨 CSS Переменные

```css
:root {
  /* Цвета */
  --color-bg: #000;           /* Черный фон */
  --color-text: #fff;         /* Белый текст */
  --color-accent: #ff6b6b;    /* Красный акцент */
  
  /* Размеры шрифтов */
  --font-size-h1: 4rem;       /* Заголовок 1 */
  --font-size-h2: 2.5rem;     /* Заголовок 2 */
  --font-size-body: 1.125rem; /* Обычный текст */
  
  /* Расстояния */
  --spacing-sm: 1rem;         /* Маленький */
  --spacing-md: 2rem;         /* Средний */
  --spacing-lg: 3rem;         /* Большой */
}
```

## 🧪 Медиазапросы

Все компоненты оптимизированы для 3 точек разрыва:

```css
/* Мобильные */
@media (max-width: 767px) {
  /* Стили для мобильных телефонов */
}

/* Планшеты */
@media (min-width: 768px) and (max-width: 1024px) {
  /* Стили для планшетов */
}

/* Десктоп */
@media (min-width: 1025px) {
  /* Стили для десктопов */
}
```

## 🔧 Flexbox компоненты

- **Header**: выравнивание logo и menu
- **Navigation**: горизонтальное расположение пунктов меню
- **Hero**: 2-колонный макет на десктопе, однолонный на мобильных
- **Footer**: распределение элементов по краям

Пример:
```css
.container {
  display: flex;

  align-items: center;              /* выравнивание вертикально */
  gap: var(--spacing-md);          /* расстояние между элементами */
}
```

## 🍔 Бургер-меню

**На мобильных (до 767px):**
- Кнопка с 3 горизонтальными линиями
- При клике на кнопку - выпадающее меню
- При клике на пункт меню - закрывается
- При клике вне - закрывается
- При resize до 768px - закрывается

**На планшетах и выше:**
- Бургер скрывается
- Показывается горизонтальное меню

## 🎨 Кастомизация

### Изменить цвета:
Отредактируйте CSS переменные в `css/variables.css`:
```css
:root {
  --color-accent: #ff6b6b;  /* Измените красный на другой цвет */
}
```

### Изменить размеры шрифтов:
```css
:root {
  --font-size-h1: 4rem;      /* Измените размер заголовков */
}
```

### Изменить точки разрыва:
Отредактируйте медиазапросы в CSS файлах (например, в header.css):
```css
@media (max-width: 800px) {  /* Измените 767px на 800px */
  /* стили для мобильных */
}
```

## 🐛 Частые проблемы

**Q: Бургер не видно на мобильных**
A: Если ширина > 767px, бургер скрыт. Используйте DevTools Device Mode.

**Q: Текст слишком маленький на мобильных**
A: Измените --font-size-body в variables.css см. RESPONSIVE_DESIGN.md

**Q: Меню не закрывается**
A: Проверьте что JavaScript активирован и нет ошибок в консоли.
